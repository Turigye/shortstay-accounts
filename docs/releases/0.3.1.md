# Short-Stay Accounts 0.3.1

## Improvements

- New Short-Stay Accounts application icon for macOS and Windows.
- Spacious booking drawer with consistent form controls and clearer labels.
- Global payment history table with customer, unit, account, reference, and receipt access.
- Six-month Today chart comparing collections, expenses, and occupancy.
- Guided property-investment entry for land, buildings, renovations, owner capital, and loans.
- Corrected toolbar alignment and payment-account settings spacing.
- Expanded automated visual checks across 1440x900, 1280x720, and 1024x640.

Installing this update preserves the existing encrypted business file, password, profiles, bookings, payments, and reports.
