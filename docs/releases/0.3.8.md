# Short-Stay Accounts 0.3.8

## Improvements

- Redesigned saved payment receipts with professional PDF formatting.
- Fixed receipt styling being blocked during print and PDF export.
- Standardized table sizing and numeric alignment across the application.
- Improved populated asset, staff, referral, payment, expense, and loan tables.

## Windows

Use the installer first. If installation is blocked, extract the portable ZIP and open `Short-Stay Accounts.exe`. Both retain the existing local business data.
