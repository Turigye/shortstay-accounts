# Short-Stay Accounts 0.3.0

## Added

- Local Admin and restricted Editor profiles.
- Editor access to active bookings, incoming payments, and receipt printing.
- Printable and reprintable payment receipts with guest, stay, balance, and operator details.
- Admin user management for creating, editing, resetting, deactivating, and reactivating Editors.
- Updated in-app guidance and beginner handbook.

## Improved

- Profile passwords are salted and hashed inside the encrypted database.
- The database unlock secret can use operating-system protected storage.
- Booking and payment audit records identify the active operator.
- Desktop layouts were reviewed at 1440x900, 1280x720, and 1024x640.

## Upgrade Safety

The schema migration preserves existing business data. Historic records remain valid and show a neutral attribution where no user profile existed yet. Create an encrypted backup before installing as normal operating practice.
