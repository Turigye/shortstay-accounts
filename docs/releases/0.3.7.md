# Short-Stay Accounts 0.3.7

## Staff payroll

- Pay each percentage-based staff allocation directly from the Staff screen.
- Record returned staff funds without duplicating entries under Expenses.
- Mark a role as not worked for a month, with a required reason.
- Payroll payments and returns now appear correctly in cash flow.

## Reliability and polish

- Fixed fixed-asset register column alignment and description wrapping.
- Added a portable Windows ZIP alongside the standard installer.
- Existing encrypted data and passwords remain in the same local application-data location.

## Windows update

1. Create an encrypted backup in Settings.
2. Try `Short-Stay.Accounts-0.3.7.Setup.exe`.
3. If installation fails, extract `short-stay-accounts-win32-x64-0.3.7.zip` and open `Short-Stay Accounts.exe`.

The portable version uses the same existing local business data as the installed version.
